# Кустик 🪴

Мобильное (телефон-first) приложение для поддержания дома в чистоте небольшими ежедневными действиями. Реализация дизайн-хэндоффа «Кустик» на **React Native + Expo (TypeScript)**.

Ядро — гибкий движок «что пора сегодня»: у каждой задачи есть комната, частота и дата последнего выполнения; приложение считает «назрелость» и показывает самое запущенное сверху. Эмоциональное ядро — живой питомец-растение, который цветёт по мере чистоты дома. Поверх — duolingo-подобная мотивация (искры, серия, уровни, достижения) и режимы один / вдвоём / семья с честной ротацией дел.

## Запуск

```bash
cd app
npm install
npx expo start          # затем i (iOS), a (Android) или w (web)
```

- **iOS / Android:** Expo Go или dev-build.
- **Web:** `npx expo start --web` (для проверки макета).

## Стек

- **Expo SDK 57**, React Native 0.86, React 19.
- **react-native-svg** — растение, иконки, кольцо таймера, превью горшков.
- **@react-native-async-storage/async-storage** — локальное сохранение (бэкенд для MVP не нужен).
- **@expo-google-fonts/nunito + caveat** — типографика (Nunito для UI, Caveat для рукописных акцентов).
- **expo-linear-gradient**, **react-native-safe-area-context**.
- Анимации — встроенный `Animated` (покачивание/«поп» растения, конфетти, выезд листов, докрутка искр, схлопывание карточек).

## Движок (формулы)

- `maturity = daysSince / freq` (0 — только сделано, 1 — ровно пора, >1 — просрочено).
- `freshness = clamp(1 − maturity/2, 0, 1)`.
- **Чистота дома** = среднее `freshness` × 100.
- **«Пора сегодня»** = `maturity ≥ 1`, сортировка по `maturity` ↓.
- **«На подходе»** = `0.55 ≤ maturity < 1`, топ-3.
- **Награда** = `round(5 × max(1, maturity))`; фокус-таймер +8.
- **Уровни** по числу выполненных дел: пороги `[6, 14, 26, 45, ∞]`.

На стартовых данных: **7 дел «пора»** и **чистота ≈ 61 %** (совпадает с чек-листом приёмки).

> Время в демо моделируется через `dayOffset` (кнопка «↻ промотать день вперёд · демо» на экране «Сегодня»). В проде заменить на реальные календарные даты и убрать демо-кнопку.

## Что реализовано

Три вкладки (Сегодня / Комнаты / Настройки) + оверлеи: редактор задачи, новая комната, экспресс «Гости на пороге», магазинчик (горшки/наряды), достижения (8), таймер «Фокус-уборка» (5/15/25), празднование «Дом сияет!», тост уровня. Растение-питомец на SVG реагирует на чистоту (листья/цвет/рот/цветы/свечение), уровень (аксессуары), время суток (💤 ночью) и погоду. Режимы один/вдвоём/семья с честной ротацией ответственных, фильтрами и недельным вкладом каждого. Геймификация: искры-валюта, серия с заморозкой, уровни, достижения.
